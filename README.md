# Gapush

The use of this tool is specifically when you are pushing to several repo sites at the same time.
For example, you have a tool that you are building and you have to push your most recent changes to
codeberg, github, gitlab, etc. Instead of having to make your own script automating this process I
built this tool to simply solve this problem.
